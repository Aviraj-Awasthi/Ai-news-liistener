# AI News Assistant

A simple web app that fetches and reads the latest news headlines.

You can type or speak your request, and the app will respond with news and read it out loud using text-to-speech.

## Tech Used

* Flask (Python)
* HTML, CSS, JavaScript
* NewsAPI
* Web Speech API

## How to Run

1. Install dependencies: `pip install flask requests`
2. Add your NewsAPI key in `app.py`
3. Run: `python app.py`
4. Open `http://127.0.0.1:5000`

## Note

Works best on Google Chrome for voice features.
